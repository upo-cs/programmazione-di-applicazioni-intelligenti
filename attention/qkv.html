Clicca un token per selezionarlo come query
Input: token della frase
0. Embedding
Ogni xᵢ ∈ ℝ^d_model
1. Proiezione lineare tramite tre matrici di pesi apprese
WQ matrice d_k × d_model qᵢ = W_Q · xᵢ → vettore d_k
WK matrice d_k × d_model kᵢ = W_K · xᵢ → vettore d_k
WV matrice d_v × d_model vᵢ = W_V · xᵢ → vettore d_v
Transformer originale:d_model = 512, d_k = d_v = 64, h = 8 heads
2. Compatibilità: qquery · kᵢ
Dot product tra due vettori ∈ ℝ^d_k → uno scalare per token
3. Score grezzi
αᵢ = softmax( score_i / √d_k )
Ogni αᵢ ∈ [0, 1] e Σ αᵢ = 1. Indicano quanta attenzione il token query presta a ciascun altro token.
4. Pesi di attenzione αᵢ
5. Output: somma pesata dei value
output = Σ αᵢ · vᵢ = α₁·v₁ + α₂·v₂ + ... + αₙ·vₙ ∈ ℝ^d_v
Q ∈ ℝ^d_k
K ∈ ℝ^d_k
V ∈ ℝ^d_v
x ∈ ℝ^d_model
